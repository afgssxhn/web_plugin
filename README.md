# Web Plugin

Плагин для Purpur/Paper, который показывает список онлайн игроков и информацию о сервере.

## Возможности

### Команда `/webp` (Online Players)
- Показывает список онлайн игроков
- Отображает: ник, страну (по IP), пинг
- Цветовая индикация пинга (зелёный < 50ms, жёлтый < 100ms, оранжевый < 200ms, красный > 200ms)
- Автоопределение локальных подключений

### Команда `/webe` (Server Equipment)
- Показывает характеристики сервера
- Отображает: OS, архитектуру, CPU ядра, RAM, Java версию

## Сборка

```bash
mvn clean package
```

Скомпилированный .jar будет в `target/Web-1.1.0.jar`

## Установка

1. Скопируй `Web-1.1.0.jar` в папку `plugins/` твоего сервера
2. Перезапусти сервер или используй `/reload confirm`

## Использование

**Показать игроков:**
```
/webp
```
или `/web_players`

**Показать характеристики сервера:**
```
/webe
```
или `/web_equipment`

## Требования

- Purpur/Paper 1.21+ или выше
- Java 21

## API для GeoIP

Плагин использует бесплатный API `ipapi.co` для определения страны по IP.
Для локальных IP (127.0.0.1, 192.168.x.x, 10.x.x.x) показывает "Local".
